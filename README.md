# Brainwave.io - SaaS Landing Page

A modern SaaS landing page built from Figma design using React and Tailwind CSS v4.

## Preview

Live preview: https://ta-01kg7y6kh5y6b9sbxqfas5z6mz-5173.wo-3raiv70d0fc1lmvw2xqonhmi7.w.modal.host

## Features

- Modern hero section with CTA
- Features grid with icons
- Content sections with images
- Statistics display
- Testimonials section
- Pricing cards (3 tiers)
- FAQ section
- Footer with navigation

## Tech Stack

- React 19
- Tailwind CSS v4
- Vite
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
