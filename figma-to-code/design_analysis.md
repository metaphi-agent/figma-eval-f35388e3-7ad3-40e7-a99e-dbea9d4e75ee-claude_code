## Design Tokens Identified

### Colors
- **Primary Brand**: Purple/Indigo (appears to be #5454D4 or similar)
- **Dark Navy**: Very dark blue/navy for footer and pricing section (#15182B or similar)
- **White**: #FFFFFF for backgrounds
- **Light Gray**: Very light gray for subtle backgrounds (#F7F7FB or similar)
- **Text Primary**: Very dark (almost black) #1E1E2F or similar
- **Text Secondary**: Gray for body text #6C6C80 or similar
- **Success Green**: For icons and positive elements
- **Warning Orange**: For specific icons

### Typography
- **Font Family**: Appears to be a modern sans-serif (likely "DM Sans" or "Inter")
- **Heading Sizes**:
  - H1 (Hero): ~56px, Bold
  - H2 (Section): ~40-48px, Bold
  - H3 (Cards): ~24px, Semi-bold
  - Body: ~16-18px, Regular
  - Small: ~14px

### Spacing
- Section padding: Large (80-120px vertical)
- Card gaps: ~40px
- Content max-width: ~1200px
- Inner padding on cards: ~32px

### Border Radius
- Buttons: ~8px
- Cards: ~16px
- Large images/mockups: ~12-16px

### Shadows
- Cards: Subtle shadow, likely `0 4px 24px rgba(0, 0, 0, 0.06)`
- Buttons on hover: Slight elevation

## Component Patterns

### Buttons
- Primary: Purple background, white text, rounded corners
- Ghost/Outline: Transparent background, text color, border
- Sizes: Medium (default), Large (hero CTAs)

### Cards
- White background
- Rounded corners
- Subtle shadow
- Icon + Title + Description pattern

### Navigation
- Horizontal menu
- Logo on left
- Links in center
- CTA button on right

## Interactive States Needed
- Button hover: Slightly darker shade
- Button active: Even darker
- Link hover: Color change or underline
- Card hover: Subtle elevation increase
- Form inputs: Focus states with primary color border
